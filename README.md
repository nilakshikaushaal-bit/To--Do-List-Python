# To--Do-List-Python

## Description
This is a simple console-based To-Do List application developed using Python. It allows users to manage their daily tasks through a menu-driven interface.

## Features
- Add a new task
- View all tasks
- Mark a task as completed
- Delete a task
- Exit the application

## Concepts Learned
- Lists
- Loops
- Conditional statements
- User input
- Basic Python programming

## Sample Menu
```
----- TO-DO LIST -----

1. Add Task
2. View Tasks
3. Mark Task as Completed
4. Delete Task
5. Exit
```
